# URL Support for Project Tabs

## Изменения

Добавлена поддержка URL параметров для табов на странице проекта.

## Как это работает

Теперь активный таб синхронизируется с URL через параметр `?tab=`:

```
/projects/{id}?tab=overview
/projects/{id}?tab=prompts
/projects/{id}?tab=pages
/projects/{id}?tab=domains
/projects/{id}?tab=mentions
/projects/{id}?tab=insights
/projects/{id}?tab=settings
```

## Преимущества

1. **Прямые ссылки**: Можно поделиться ссылкой на конкретный таб
2. **История браузера**: Кнопки "Назад/Вперед" работают между табами
3. **Закладки**: Можно сохранить закладку на конкретный таб
4. **SEO-friendly**: Поисковики могут индексировать разные табы

## Реализация

Изменения внесены только в `ProjectDetailPage.tsx`:

1. Добавлен `useLocation` hook
2. При инициализации activeTab берётся из URL параметра
3. Два useEffect хука синхронизируют URL и activeTab в обе стороны

## Обратная совместимость

- Если параметр `tab` отсутствует, открывается таб "overview" по умолчанию
- Все существующие ссылки `/projects/{id}` продолжают работать
- Старый функционал с кликабельными табами сохранён полностью

## Производительность

- Не добавляет дополнительных рендеров
- Минимальное влияние на производительность
- Размер бандла не изменился
